# LoanPro – Loan Management System

A simple and user-friendly web application designed to help small-scale lenders manage their loan operations efficiently. This system focuses on simplicity, speed, and real-world usability.

---

## 🚀 Overview

LoanPro helps a lender to:

- Manage clients and their information
- Record loans issued
- Track repayments easily
- Manage collateral attached to loans
- Store company/business details
- Access everything from one clean dashboard

The system is designed to work on both laptops and phones.

---

## 🎯 Purpose

This project was developed to solve real challenges faced in small loan businesses such as:

- Losing track of borrower records  
- Forgetting repayment dates  
- Difficulty managing multiple clients  
- Poor organization of financial data  

LoanPro provides a simple and structured digital solution.

---

## 🛠️ Features

- 🔐 Account-based login system
- 👥 Client management
- 💰 Loan tracking
- 📊 Repayment monitoring
- 📦 Collateral management
- 🏢 Company profile section
- 🎨 Clean and modern interface

---

## 🧰 Tech Stack

- **Backend:** Node.js  
- **Framework:** Express.js  
- **Frontend:** HTML, CSS, JavaScript  
- **Database:** Local file-based storage
